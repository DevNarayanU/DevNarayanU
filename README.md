##  Overview

This is my personal website made with the passion to learn html and css. Although one may find it to be lacking in some field, there is always some room for development, that can be improved in the upcoming months.
The website is designed with a **modern minimal aesthetic**, smooth transitions, and responsive layouts to provide a clean user experience.

It highlights:

* My projects and GitHub work
* Educational background
* Personal interests and skills
* Contact links for collaboration

---

##  Tech Stack

* **HTML5** — Structure of the website
* **CSS3** — Styling, animations, and responsiveness
* **Custom Fonts** — Jost font family
* **GitHub Pages** — Hosting
